Project E-ward

E-Ward is a full-stack web application developed for Ward 17 of Lalitpur Metropolitan City to digitally manage community services. The platform includes modules for publishing notices, managing training events, and handling user registrations. I designed the UI in Figma with a focus on clarity and accessibility, then collaborated on development using ASP.NET Core Razor Pages and MySQL. This project allowed me to bridge design and functionality to serve real-world users effectively.
